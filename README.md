# Project books-api


# Routes

| HTTP   | Endpoints        | Purpose    |
| -----  | -----------------| ---------- |
| GET    | /                | Home page  |
| GET    | /books           | Index page listing all books  |
| PATCH  | /books/update    | Update book|
| DELETE | /books/:id       | Delete book|
| POST   | /books           | Add book|

# Database

| Field | Type |
| ----- | -----|
| _id   | Object ID |
| title | String |
